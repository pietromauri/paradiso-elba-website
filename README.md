# Paradiso Elba Website

Questo è il repository per il sito web di "Paradiso Elba", un moderno appartamento per vacanze situato sull'Isola d'Elba.

## Descrizione

Il sito web è costruito con React e TypeScript, utilizzando Vite come build tool. Presenta l'appartamento, le attività nei dintorni, una galleria di immagini e una sezione per le recensioni degli ospiti.

## Caratteristiche

*   **Homepage:** Una landing page accattivante con una hero image e un'introduzione all'esperienza.
*   **Galleria:** Una galleria di immagini filtrabile per categorie (panorami, spiagge, ecc.).
*   **Componenti riutilizzabili:** Un design system basato su componenti React per una facile manutenzione.
*   **Design responsive:** Adattabile a tutti i dispositivi, dai desktop ai mobile.
